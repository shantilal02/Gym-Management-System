# Gym Management System

## Overview

Gym Management System is a web-based application developed to simplify and automate daily gym operations. The system helps gym administrators manage members, trainers, attendance, memberships, payments, schedules, and reports efficiently.

This project is designed to reduce manual work and improve management accuracy using a user-friendly interface.

---

# Features

* Member Registration and Management
* Trainer Management
* Attendance Tracking
* Membership Management
* Payment Management
* Workout Schedule Management
* User Authentication and Login System
* Dashboard for Gym Administration
* Report Generation
* Responsive User Interface

# Installation Process

## Step 1: Clone Repository

```bash
git clone https://github.com/shantilal02/Gym-Management-System.git
```

## Step 2: Open Project Folder

```bash
cd Gym-Management-System
```

## Step 3: Install Required Packages

```bash
pip install -r requirements.txt
```

## Step 4: Run Server

```bash
python manage.py runserver
```
